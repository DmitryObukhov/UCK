Informationen zur Installation


Wichtig: Die Paketsammlung im Swissremix belegt rund 6 GB Festplattenspeicher, für die Installation empfohlen sind 10 GB oder mehr.

Die Installation eines Linux Komplettsystems ist deutlich einfacher und geht massiv schneller vor sich, als man das von den etablierten Systemen her kennt. Auch einem Anfänger gelingt die Installation meist mühelos, insbesondere wenn die gesamte Festplatte für Ubuntu genutzt werden kann.

Etwas heikler wird es, wenn das vorhandene System verkleinert werden soll, damit es sich den Festplattenplatz mit dem neuen Ubuntu teilen kann. Auf jeden Fall ist vorher eine Systemsicherung zu erstellen. Der Vorgang nennt sich 'Partitionieren' und auch für diesen Fall gibt es eine Benutzerführung. Wenn man genau den Anweisungen am Bildschirm folgt, gelingt die Installation problemlos und man kann über ein Startmenu zwischen Ubuntu und dem vorherigen System wählen. Die empfohlene Mindestgrösse für ein System mit Ubuntu SwissRemix ist 10 Gigabyte.

Es gibt viele Varianten und Vorlieben, wie man das System einrichten kann. Wir empfehlen, mindestens die persönlichen Daten in eine eigene Partition (Bereich der Festplatte) zu legen. Wer dazu genaueres erfahren will, informiert sich via Fachliteratur, wendet sich an einen Linux Spezialisten (in der Nähe oder an einem Szene Treff) - oder kontaktiert professionellen Support.

Ebenfalls empfohlen, vor allem für alle Arten von tragbaren Geräten, ist die Verschlüsselung der Festplatte oder mindestens der Datenbereiche.  Die dafür benötigte Software cryptsetup und lvm2 ist auf dem SwissRemix enthalten, der Vorgang erfordert allerdings etwas Übung mit Linux Systemsoftware. Die Anleitung dazu gibt es z.B. im Wiki von Ubuntuusers.de. Auch hier können Ubuntu Fans und Profis weiterhelfen - eine elegante Möglichkeit bietet das Werkzeug Gitso, welches man für die Fernunterstützung verwenden kann. Gitso ist im Menu Anwendungen/Internet zu finden, für die Nutzung sind lediglich die Angaben des Supporters nötig.

Wer schliesslich die Möglichkeiten ausschöpfen möchte, konfiguriert ein Software RAID. Und zwar ohne Einschränkungen und mit denselben Mitteln, wie es die Profis tun.

Besonders wichtig bei RAID und verschlüsselten Festplatten ist die Sicherung von System und Benutzerdaten. Auf dem Ubuntu SwissRemix gibt es dafür eine Reihe von Backup-Lösungen  

Wichtig: Ubuntu SwissRemix vervollständigen

Bitte führen Sie nach der Installation den Script "Installation vervollständigen" auf dem Desktop aus. Sie benötigen dafür eine Internetverbindung.

Obwohl automatisiert lösbar, ist dieser Schritt aus rechtlichen Gründen notwendig. Die Installation auf der DVD enthält enorm viele Softwareprodukte, welche man alle frei benutzen und verteilen darf. Es gibt aber auch häufig genutzte Komponenten mit rechtlichen Einschränkungen, darum hier ein paar Erklärungen zu den Rahmenbedingungen:
  • Eine wichtige Eigenschaft von Software ist der Lizenzvertrag des Herstellers. Er regelt die Rechte des Benutzers, bestimmt aber auch die Bedingungen zum Einsatz und definiert gewisse Einschränkungen, zum Beispiel ob oder wie oft das Produkt kopiert werden darf. Ohne Lizenzvertrag kommt das Urheberrecht zur Geltung, welches in der Regel den Schwerpunkt auf den Schutz von geistigem Eigentum legt.
  • Die alternative Betrachtungsweise geht davon aus, dass Zugang zu Wissen frei sein soll - und dass es sich bei Software um 'niedergeschriebenes Wissen' handelt. Darum wurden für freie Software besondere Lizenzvereinbarungen entwickelt, welche z.B. weniger Einschränkungen bezüglich Verwendung machen. Die Weitergabe und das Kopieren ist ausdrücklich erlaubt - ebenso wie die Installation auf beliebig vielen Geräten. Man darf die Software sogar verändern - und erst an der Stelle kommen auch besondere Pflichten ins Spiel, nämlich, dass man die Veränderungen wieder der Allgemeinheit zur Verfügung stellt. 
  • Auf einer DVD wie dem Ubuntu SwissRemix darf aufgrund der Rechtslage nur Material vorhanden sein, welche via Lizenz freigegeben wurde. Das trifft auf Ubuntu (System, Dokumentation und Anwendungen) zu - und auch auf die vielen Zusatzprogramme, Informationen und Medien.
  • Andere Programme - wir nennen sie in dem Zusammenhang 'unfreie Software' - mögen zwar gratis sein, dürfen aber nicht verteilt werden oder sind nur zum nicht-kommerziellen Gebrauch bestimmt. Beispiele: Adobe Reader, Skype, Google Earth, Codecs, Treiber oder Firmware. Der benutzer muss diese also selbst herunterladen. Mehr dazu unter Informationen dazu unter http://www.ubuntu.com/ubuntu/licensing
  • Um trotz dieser Einschränkungen eine möglichst komfortable Installation zu erlauben, können ein paar empfohlene Komponenten aus den vordefinierten Quellen über die Funktion 'Installation vervollständigen" eingerichtet werden. Es handelt sich um die Pakete ubuntu-restricted-extras, libdvdcss2, kubuntu-restricted-extras und non-free-codecs.
  • Besonderes: Innerhalb von Firmen und juristischen Personen gelten besondere Reglungen! Es ist dort möglich, einen Remix mit wenig aufwand inklusive proprietären Treibern oder nicht-freier Software zu bauen und zu verteilen. Näheres dazu erfahren Sie bei einem Dienstleister in der Nähe.

Hinweise zur Nutzung von Ubuntu

Wie installiert man weitere Software?

Grundsatz 1: man bezieht Software heutzutage nur von einer vertrauenswürdigen Quelle (Repository), möglichst via Internet. Installationen ab CD sind dank Breitbandverbindungen nicht mehr zeitgemäss und Software von den Downloadportalen ("Abzockerseiten") ist schwer zu trauen. Man erspart sich also  die mühsame Suche nach Programmen und reduziert die Risiken von schädlicher Software oder durch unseriöse Anbieter. Dank der Softwarequelle sind Betriebssystem, Treiber und die installierten Programme laufend aufeinander abgestimmt und können in einem Rutsch aktualisiert werden. Unter Ubuntu heissen die Schnittstellen zur Software-Quelle Software Center (im Menu Anwendungen) oder Synaptic (im Menu System/Systemverwaltung). Es ist möglich, zusätzliche - auch eigene, z.B. firmeninterne - Quellen zu aktivieren, Stichworte dazu: PPA, Fremdquellen

Grundsatz 2: man holt sich keine ausführbaren Linux Installationsprogramme auf den PC. Ausser man weiss, was man tut. Sofern die Quelle vertrauenswürdig ist, lädt man von dort eine *.DEB Datei (z.B. auf den Desktop) und lässt diese z.B. mit Doppelklick oder via rechte Maustaste durch die Softwareverwaltung (Gdebi) einrichten. Ein paar Beispiele befinden sich auf der DVD unter /options. Nachteil dieser Variante: Die Abhängikeitsprüfungen funktionieren nur teilweise und das Paket wird später nicht automatisch aktualisiert. Ein Beispiel: Skype - ist aktuell in keinem Repository erhältlich und muss von der Herstellerseite geladen werden: http://www.skype.com/intl/de/download/skype/linux/choose
Immerhin gibt es eine *.DEB Datei (und auch *.RPM Versionen für die zweite grosse Linux Familie).

Hände weg: Nicht empfohlen bzw. den Profis vorbehalten ist die Verwendung von Installationsprogrammen. Trotzdem kann man einem solchen Werkzeug begegnen (z.B. Programme zum Ausfüllen der Steuererklärung) und lässt die Installation dann ausnahmsweise zu.  Abhängigkeitsprüfungen funktionieren dann kaum und es kann sein, dass man die Software nicht restlos deinstallieren kann. Während man dieses Verhalten von anderen Plattformen kennt, ist das für Linux System untypisch. Es versteht sich so von selbst, dass vom Starten von *.EXE Dateien dringend abgeraten wird, auch wenn erstaunlich viele Windows Programme unter Linux laufen.

Ausnahmen: Es gibt auch gute kommerzielle Linux Software für den professionellen Einsatz, welche nicht über eine Softwarequelle oder ein *.DEB Paket installiert wird. Beachten Sie in diesem Fall die Empfehlungen des Herstellers.

Von Windows auf Linux umsteigen

Der Ubuntu SwissRemix liefert eine Fülle von ausgetesteter und vorinstallierter freier Software für den täglichen Einsatz. Auf dem Internet findet man Anleitungen und Hilfen, wie der Umstieg am besten gelingt. Achtung: Es sind die geheimniskrämerischen, monopolistisch angehauchten und proprietäreren Produkte, welche bei der Umstellung etwas schwieriger sind und welche laufend tückischer werden - und auf die man auf Dauer sowieso lieber verzichtet.   

Für fast jedes Szenario - zumindest im Bürobetrieb - ist eine Lösung zu finden und je nach Programm geht man den einen oder anderen Weg:

  • sich mit den Alternativen befassen und auf ein geeignetes freies Programm umstellen
  • das unfreie Programm mit einem freien Programm ersetzen und mit den original Daten weiterarbeiten
  • die Daten umformatieren und mit einem freien Programm weiterbearbeiten
  • die alten Daten nur noch mit einem Viewer betrachten und auf ein neues Format umsteigen
  • das Windows Programm unter Wine benutzen
  • das Windows Programm in einem virtuellen lokalen PC oder via Terminal Server starten
  • statt dem lokalen Programm eine Weblösung verwenden

 
Vielleicht aber möchte jemand freie Software auf dem jetzigen Betriebssystem probieren? Die Windows Versionen einiger bekannter Vertreter von freier Software befinden sich auf der DVD unter /programs. Gerade Thunderbird für Windows ist ein unverzichtbares Werkzeug, wenn man Emails und Kontaktdaten aus den Fesseln einer verbreiteten geheimen Technologie retten will.

Hardware talk

Die Linux Gemeinschaft ist stolz auf die die riesige Liste von Hardware, welche auf Anhieb mit Linux funktioniert. Es ist das Ergebnis der gemeinsamen Arbeit von Standardisierungsorganisationen, Tausenden von Freiwilligen, der Linux Distributoren und auch der Hardware Anbieter.

Wenn heute eine Hardware nicht unterstützt werden kann, z.B. weil sie sehr neue Komponenten enthält, liegt dies oft leider an nicht vorhandener Unterstützung oder fehlender Offenlegung durch den Hersteller. Es ist deren Recht und deren Entscheid, ihr geistiges Eigentum zu schützen - und nicht ein Manko von Linux oder der Community Arbeit. Als Kunden haben wir die Möglichkeit, die Hersteller um Linux Unterstützung für ein bestimmtes Gerät zu bitten - oder Hardware zu wählen, für welche die gesuchte Unterstützung vorhanden ist.

In Mittelweg ist - wie oben schon beschrieben - die Verwendung von 'unfreier' Software. 'Unfrei' kann auch hier heissen, dass die Software zwar gratis genutzt werden kann, aber nicht auf einer DVD verbreitet werden darf. Oder es gibt andere Einschränkungen, z.B. keine 64-Bit Version. Verschiedene Hersteller bieten solche 'closed-source' Unterstützung - dann kann eventuell über die 'Hardware' Option im System Menu von Ubuntu solche Software nachgeladen werden (beispielsweise von einigen Grafikkarten). 

Andere Geräte - z.B. USB TV Adapter oder Audio Karten - benötigen eigene Firmware - Programmcode, welcher durch den Treiber auf die Hardwarekomponente geladen wird. Meist ist Verteilung von Firmware nicht erlaubt, man muss diese entweder separat beschaffen - oder aus den mitgelieferten Treibern extrahieren. Anweisungen für die eigene Hardware findet man auf dem Web, die Firmware wird dann nach /lib/firmware kopiert.

Die Windows Unterstützung befindet sich auf der Original-CD zum Gerät oder muss vom Hersteller bezogen werden.  Ein hierzulande verbreitetes Gerät, welches mit Ndiswrapper läuft, ist die Globetrotter PCMCIA UMTS and WLAN Karte (eingesetzt als "Mobile Unlimited PC Card"): Einfach mit der Anwendung 'Windows LAN Treiber' die Datei *.inf öffnen (im Falle der Globetrotter Karte: den Treiber findet man bei der Internetsuche nach marvel8335XP.tar.gz).


Die Idee hinter dem Swiss Remix?

Der Verband /ch/open (The Swiss Open Systems User Group) und die Schweizer Free Software Communities unterstützen die Verbreitung von freier Software - Free and Open Source Software (FOSS). Free & Open Source Software eignet sich heute hervorragend für die Nutzung auf Desktops und Laptops - und die  Swiss Remix DVD soll bei der Verbreitung helfen.
 
Dank der Offenheit und der Freiheit von FOSS ist es jedem erlaubt, diese Software nicht nur zu nutzen, sondern auch zu verteilen - oder einen eigenen Remix zu bauen. Das ist speziell interessant für Schulen, Firmen oder andere Organisationen.

Die DVD demonstriert die Vielfalt und Funktionalität der freien Software. Sie kann auch als Installationsquelle genutzt werden - wegen dem grossen Softwareumfang gerade für Systeme ohne (Breitband-)Internetverbindung geeignet. 

Die Sammlung enthält nützliche Werkzeugen für Büro, Bildverarbeitung, Multimedia etc, einige länderspezifische Voreinstellungen, Werkzeuge für Administratoren und die verbreiteten drei Desktops (Ubuntu, Kubuntu und Xubuntu) - das alles auf einem einzigen Installationsmedium. Mit dabei sind viele Ausbildungsprogramme der Edubuntu Distribution.

Soweit verfügbar, befinden sich die Sprachdateien für Deutsch, Französisch, Englisch und Italienisch auf der DVD sowie die zum Zeitpunkt der Erstellung erhältlichen Softwarekorrekturen.

Die oben genannte freie Weitergabe gibt es nicht nur für Software und Programme. Begeisterte Leute  auf der ganzen Welt schöpfen Kunstwerke und geistiges Eigentum,  welches frei verfügbar ist. Es gibt Fotos, Videos, Texte - z.B. unter der "Creative Commons Lizenz". Die Weitergabe ist ausdrücklich erlaubt - meist unter Namensnennung und unter derselben Lizenz. Ein paar Beispiele sind auf der DVD unter /extras zu finden, die beiden oben genannten Bücher zu Ubuntu Linux gehören ebenfalls dazu.

Paketauswahl

Die umfangreiche Paketauswahl kam zustande durch Diskussionen mit Benutzern, Lektüre von Blog Einträgen, unseren eigenen Vorlieben und Hinweisen aus der Schweizer Community. Eine Liste der Ergänzungen wird zusammen mit den ISO Dateien der DVD auf dem Web zur Verfügung gestellt. Bevor man sich also auf die Suche nach Zusatzsoftware macht, einfach schnell prüfen, ob das passende Programm nicht schon installiert ist ...

Die DVD enthält die Oberflächen Gnome, KDE und XFCE. Standardmässig ist Gnome eingestellt, welches von etwa 80% der Ubuntu Benutzer bevorzugt wird, und die Paketauswahl fokussiert auf Gnome. KDE und XFCE sind in einer Grundausstattung vorhanden und können durch den Benutzer nach seinem Gusto erweitert werden.

Aus verschiedenen Gründen wurde mit dem SwissRemix am Verhalten von Ubuntu nichts (bewusst) geändert und alle Programme erhalten die Standardeinstellungen.

Why a Swiss Remix? 

The Swiss Open Systems User Group (/ch/open) and the Swiss Open Source communities want to spread the word about Free Open Source Software (FOSS) in Switzerland - by emphasizing its readyness for everyday use, in particular for desktops and laptops. Thanks to the freedom and openness of  FOSS it is legally possible for anyone - schools, companies, organizations - to not just use and copy the software and media, but also to create an own distribution based on Ubuntu.

One main purpose of the DVD is to serve as demonstration of the variety and functionality of current free software. The DVD can be used as installation source for systems not connected to the internet. The intention was to bundle additional useful software (office tools, picture editing utilities, troubleshootingtools) and some Swiss settings, and also to make the three major Ubuntu flavours (Ubuntu, Kubuntu, Xubuntu) available and usable on the same installation. Educational programs from Edubuntu are packed into the SwissRemix, too.

All language components for English, German, French and Italian support are ready on the DVD, both in live mode (running from DVD or USB-stick) and when installed on a hard disk. Language and keyboard are easily changed when booting or when logging in. The DVD includes updates available at the date of production.


 Freedom to share is not limited to software. Enthusiasts around the world create artwork and  intellectual property which is freely available. Look for media and documents under a  „Creative Commons License“ or similar. These allow free copying - usually under the same licence - and in some cases also derivative works. Check the /extras to find some books, music, videos and talks.


Package selection

We were inspired by software discussions on many blogs, we have observed our own preferences and had several discussions with members of the Swiss Ubuntu community. The full list of additions is maintained on the web along with the ISO files of this DVD.  In case you are looking for additional software (PDF Viewer, ZIP, Graphics Editing Software, text editor for power users, ...),  check if it's not already there ;-)


It was decided not to modify any default behaviour of the underlying software,  additional programs come 'as-is' with no particular customization and for current bugs we don't provide known workarounds. This ensures maintainability of the systems and allows users to apply procedures which work on a standard system.
 

Note for Windows Users switching to Linux

Although the Linux platform offers a great deal of tested and preinstalled free software for everyday use, many people initially prefer to try Free Open Source Software on their current plattform.  A very small selection of popular software is available on the DVD under  /programs.

Hardware talk

The linux community is proud of the huge amount of hardware that runs right out of the box. It is the result of combined efforts by organizations defining open standards, by thousands of volunteers, by the linux distributors and hardware companies. If a piece of hardware does not run with free software today, it is usually due to limitations imposed by hardware manufacturers in order to guard their intellectual property. It's their right and their decision, that's fine, and it's not Linux to blame for it. As a consumer, we have the option to call the manufacturer for support for our particular device, or to choose hardware which is known to be well supported by Linux.

Another possibility is to use your hardware with non-free software components developed and offered by the hardware manufacturers. As one example you may use the "Hardware" option in the system menu of Ubuntu to detect devices supported by proprietary drivers (example: Nvidia). Such software is legally protected from being distributed and may not provide all the advantages of open source software (missing 64-bit capabilities, for example), but can be downloaded for free.

Other devices like USB TV sets may require firmware to be copied  to /lib/firmware. Find installation notes for your device on the web.

'Windows LAN drivers' (ndiswrapper), as included in the SwissRemix in the System menu, uses Windows drivers for a selection of network or wireless cards.  The windows driver itself must be copied from original CD or manufacturers site. One example of a card that runs quite well with ndiswrapper is the Globetrotter PCMCIA UMTS and WLAN Card (Mobile Unlimited PC Card): Just point 'Windows LAN drivers' to the *.inf file (for this particular card: find marvel8335XP.tar.gz on the web first).

Frequently asked questions

Q: Can I use the DVD for an upgrade?

A: The recommended way for upgrades is using the „Update Manager“, available on all current Ubuntu Systems. Ubuntu 8.04 systems have to be upgraded to 8.10 first, then to 9.04 before upgrading to 9.10. Please consider the following two options: The next version of Ubuntu, coming out in April 2010, is a Long Term Support edition (LTS) and will feature updates from Ubuntu 8.04 Hardy Heron (the latest LTS version). Or you might use the Alternate / Text installer - also available from the boot menu of the SwissRemix - to upgrade or to fix a broken installation.

Q: is there a 64-bit or Netbook Version of Ubuntu 9.10 SwissRemix d-f-i-e available, too?

A: No, our goal was to build a well equipped 32-bit edition, bundled with some free media to fill about one DVD. As time allows, there occasionally will be a 64-bit ISO built available from www.teamubuntu.ch as download only. We are evaluating possibilities to promote existing installations - including special purpose editions - to SwissRemix clients. PowerPC Editions seem to be worth thinking about as those nice G5 Macs are not supported anymore by the latest Apple OS ...

Q: I can't find the KDE 3.5 desktop

A: SwissRemix is built on Ubuntu 9.10 which uses current edition of KDE.


Q: Where can I get Support for the Ubuntu SwissRemix? 

A: Ubuntu SwissRemix is built with original Ubuntu packages. Please get advice from regular Ubuntu Wiki or Forums. You may visit our website to find a list of contacts and resources

http://www.ubuntu.ch


Q: Is WINE – the Windows emulator – part of SwissRemix?

A: Our original position was to omit WINE:
a) Software in the Windows environments can't be managed by a package manager nor is there a trustworthy download source. As users continue to install software in Wine, the Windows subsystem gets unmanageable, cluttered and inconsistent as known from regular Windows PC installations.

b) Although WINE itself is free and able to run a variety of Windows executables, many programs require additional libraries, fonts or subsystems which are not freely available. A valid Windows license is required for certain libraries.

Yet, we decided to include Wine and further packages (Virtualization components, CHM readers) to ease transition or integration into existing environments. Concerns as mentioned above still apply.

Q: I have already installed Karmic Koala. Can I put the SwissRemix on top?

A: Yes, you can. Soon. Look for step-by-step instructions to be found on our website shortly. The impatient might send a note to bugs@teamubuntu.ch  

Q: Why are there different terminals in the menu or several chat tools?

A: The goal of the Remix was to provide a selection of recommended software and the option to switch between complete desktops from Ubuntu, Kubuntu and Xubuntu. Each environment adds its own and optimized utilities to the shared application menu system, giving users the opportunity to evaluate and select the tool they prefer. 
For example, the selection of chat tools includes Pidgin, Quassel, X-Chat, and Kopete, amsn. Apart from their origin of desktop (likely to be visible in their 'About' or 'Help' Menu), they have different characteristics to suit different user requirements.

Q: I was told that the SwissRemix is a compelte Desktop system, but I can't find Adobe Reader, nor Quicktime Player nor other of my Favourites.
A: Bear in mind the functionality provided by the Tool you're looking for. Ubuntu has a generice document viewer which is invoked when you doubleclick a PDF. Many multimedia formats brought to you by Quicktime can be played by Linux standard tools. You will find lists of alternatives on the web.

Q: I have installed Ubuntu SwissRemix without any errors. Invoking 'Finish Installation/Installation abschliessen" I see the error message "xyz is a virtual package"
A: just run one regular update of Ubuntu System. This usually cleans up the error.
