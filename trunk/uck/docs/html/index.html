UCK - Ubuntu Customization Kit

UCK is a tool that helps you customizing official Ubuntu Live CDs (including Kubuntu/Xubuntu and Edubuntu) to your needs. You can add (almost) any package to the live system, for example language packs, or applications.

Features

  • Create bootable LiveCD with predefined languages based on original Ubuntu/Kubuntu live CD using a simple wizard GUI
  • Build live CD with special features using scripts. It is possible to customize root filesystem (for example install/remove packages), ISO contents (add/remove docs, change names) and initrd (add modules to boot, change boot sequence)
  • Harness the power of the UCK scripts with the uck-flow GUI.

Project details

UCK is developed and distributed under the GPL v3 license.

Source code is maintained in SVN at http://sourceforge.net/projects/uck.

As the project is mainly geared towards Ubuntu, packaging, bug tracking and general support happens on Launchpad https://launchpad.net/products/uck. This is also why we only rarely release updated source archives on SourceForge.

Up-to-date stable packages (src and binary) for recent Ubuntu releases may be downloaded from https://launchpad.net/~uck-team/+archive/uck-stable

Snapshot releases (src and binary) are available from the PPA at https://launchpad.net/~uck-team/+archive/uck-unstable.

There are two mailing lists for UCK on SourceForge, both currently with very low traffic and only relevant for the UCK development team.
