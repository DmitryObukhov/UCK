Ubuntu Customization Kit

The remastering backend

UCK has several utility scripts to allow you a flexible remastering, here you can find the complete list (in logical order):

  • uck-remaster-unpack-iso
  • uck-remaster-unpack-rootfs
  • uck-remaster-unpack-initrd
  • uck-remaster-chroot-rootfs
  • uck-remaster-prepare-alternate
  • uck-remaster-finalize-alternate
  • uck-remaster-remove-win32-files
  • uck-remaster-pack-initrd
  • uck-remaster-pack-rootfs
  • uck-remaster-pack-iso
  • uck-remaster-clean
  • uck-remaster-clean-all

uck-remaster-unpack-iso

Usage: 

# uck-remaster-unpack-iso iso-file.iso [remaster-dir]

uck-remaster-unpack-rootfs

Usage: 

# uck-remaster-unpack-rootfs [remaster-dir]

uck-remaster-unpack-initrd

Usage: 

# uck-remaster-unpack-initrd [remaster-dir]

uck-remaster-chroot-rootfs

Usage: 

# uck-remaster-chroot-rootfs [remaster-dir] [what-to-execute]

uck-remaster-prepare-alternate

Usage: 

# uck-remaster-prepare-alternate [remaster-dir]

uck-remaster-finalize-alternate

Usage: 

# uck-remaster-finalize-alternate gpg-key-id [remaster-dir]

uck-remaster-remove-win32-files

Usage: 

# uck-remaster-remove-win32-files [remaster-dir]

uck-remaster-pack-initrd

Usage: 

# uck-remaster-pack-initrd [remaster-dir]

uck-remaster-pack-rootfs

Usage: 

# uck-remaster-pack-rootfs [remaster-dir] [-c|--clean-desktop-manifest]

uck-remaster-pack-iso

Usage: 

# uck-remaster-pack-iso [iso-file-name.iso] [remaster-dir] [-h|--hybrid] [-g|--generate-md5] [-a|--arch=x86,x86_64,ia64,ppc]  [-d|--description=CD_description]

uck-remaster-clean

Usage: 

# uck-remaster-clean [remaster-dir]

uck-remaster-clean-all

Usage: 

# uck-remaster-clean-all [remaster-dir]
